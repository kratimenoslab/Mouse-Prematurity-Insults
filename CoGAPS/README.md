# CoGAPS Analysis

CoGAPS (Coordinated Gene Activity across Pattern Subsets) analysis code.

## Overview

cerebellum_cogaps_mouse_n15.R : Run script for CoGAPS factorization

cogaps_mouse_n15_res.ipynb : Analyze CoGAPS patterns; pattern visualizations and metadata correlations

cogaps_mouse_n15_GSEA.Rmd : Run GSEA on CoGAPS gene weights
